Random Choice Order

Short little plugin that randomizes the order of choices. Great for trivia questions or any situation where you don't want players to memorize the position of choices.

The plugin overwrites the functions Window_ChoiceList.prototype.makeCommandList and Window_ChoiceList.prototype.callOkHandler, so make sure to put this plugin above plugins that use these functions.

XP Theme01

An arrangement of the RPG Maker XP RTP's Theme01, looped and ready-to-use in OGG format. Overall, not too much has changed from the original MIDI as aside from new instruments and sounds. I mostly messed with instrument choices, like adding a bassoon to complement the oboe, and a piano in the second half.

I don't need to be credited for this song, but you need to own RPG Maker XP as the original composition is from the RTP.
